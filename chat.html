Intrucciones

1. Espera unos segundos para que cargue el applet del Chat, te daras cuenta cuando te salga una ventana con un mensaje donde te pide confirmacion para aceptar el applet debes darle si a este mensaje para descargarlo, de no aparecer dicha ventana es por que no tienes instalado en tu pc el Kit Compilador de Java (ver punto 2)
2. Debes tener instalado en tu pc el Kit Compilador Java de no ser asi debes bajarlo de la siguiente direccion: http://java.sun.com/downloads/
3. Una vez Cargado el Applet se abrira una ventana donde debes ingresar tu Nick y debes elegir el canal a donde ingresaras.
4. A chatear :)
